# Shopping List Manager

A simple command-line Python application that helps users manage a shopping list.

# Features
* Add items to the shopping list
* Remove items from the shopping list
* Display all items in the shopping list
* Show the total number of items
* Exit the application

# Technologies Used
* Python 3

# How to Run
1. Make sure Python 3 is installed on your computer.
2. Download or clone this repository.
3. Open a terminal in the project folder.
4. Run the program using:

```bash
python shopping_list.py
```

## Example
```
Welcome to online shopping...

Please select any option below
1. Add item

3. Display items
4. Exit

Enter your choice: 1
Enter the item you want to add: Milk
Milk is added to shopping cart

Enter your choice: 3
```
